Stencila Style Guide

About

Welcome to the Stencila Style Guide.

You are probably viewing the current master version of this guide at https://stencila.github.io/style/guide/. If you want to go back to a previous version use the URL pattern https://cdn.rawgit.com/stencila/style/[ref]/index.html where [ref] is a tag e.g. 0.1 or commit 6b369c3dac104430a5c9827799bb2275a066c6e0.

This guide has examples of UI elements shown with a yellow-border. Click on the 'example' label at the bottom to see and/or copy the HTML for the example. In addition, there are separate pages for larger examples.

Want to contribute? Great, see the README!

Colours

These colours are not applied directly by developers but instead using modifers like is-primary (blue).
